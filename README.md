# Environment-Monitoring
It is Iot based project which is measure the temperature, humidity, soil moisture and light intensity

🌱 Environment Monitoring System using ESP8266

This project is an IoT-based Environment Monitoring System built using ESP8266. It collects real-time environmental data such as temperature and soil moisture and sends it wirelessly for remote monitoring.

🚀 Features
🌡️ Temperature monitoring
💧 Soil moisture detection
⚡Light intensity using ldr
📡 Wi-Fi connectivity using ESP8266
📱 Remote monitoring (mobile/web dashboard)
⚡ Low-cost and efficient system

🛠️ Components Used
ESP8266 (NodeMCU)
DHT Sensor (Temperature & Humidity)
Soil Moisture Sensor (plate-based)
Jumper Wires
Power Adapter / USB Supply
